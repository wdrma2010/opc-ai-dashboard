# OPC Singular AI Dashboard

Enterprise-grade AI data visualization and analytics platform.

## Features
- Real-time data dashboard
- Multi-language support (9 languages)
- RESTful API
- WebSocket data streaming

## Quick Start
```bash
cp .env.sample .env
# Edit .env with your configuration
npm install
npm start
```

## Deploy
Deploy to Zeabur with one click via Docker Image.
